## Deskripsi

Website Absensi SMKN 8 Medan merupakan sistem absensi berbasis web yang digunakan untuk mengelola data siswa, absensi harian, izin, pengumuman, jadwal, dan berbagai fitur administrasi sekolah.

## Teknologi

* Laravel
* MySQL
* Bootstrap
* JavaScript

## Fitur

* Login Admin dan Siswa
* Absensi Berbasis Lokasi
* Manajemen Pengumuman
* Manajemen Event
* Pengaturan Jam Kerja
* Pengelolaan Data Siswa
* Dashboard Monitoring
